Vtb
===

Temperature converter


Build and run tests:
--------------------
$ `mvn verify`

Launch:
-------
$ `mvn tomcat7:run-war-only`

Open [http://localhost:8080/Vtb](http://localhost:8080/Vtb)

Used technologies:
------------------
Back-end: Servlet, JAX-WS, CDI, JSON, JUnit, EasyMock, HttpComponents

Front-end: HTML/CSS, jQuery
